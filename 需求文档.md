# 资产库项目 - 需求文档

## 项目概述

| 项目名称 | 资产库管理系统 |
|----------|----------------|
| 项目性质 | 纯前端展示型页面 |
| 项目功能 | 车站设备资产管理、材质编辑 |

---

## 页面结构

### 入口页面
- `资产库/车站资产.html` - 车站资产管理主页面
- `佳都资产库/资产库-设备管理.html` - 佳都风格设备管理

### 左侧菜单
- 设备模型（15个分类）
  - 通风空调与采暖系统
  - 通信系统
  - 给水与排水设备
  - 自动售检票系统
  - 乘客信息系统
  - 供电
  - 主变电系统
  - 站台门系统
  - 环境与设备监控系统
  - 电梯客运系统
  - 综合监控系统
  - 安防系统
  - 门禁系统
  - 灭火系统
  - 防雷与接地
- 材质管理
- 车站资产
- 系统管理

### 设备管理 Tab
- 树形结构左侧面板
- 设备列表表格
- 操作列：详情、预览、编辑、下载、删除

### 设备编辑页面
- `设备编辑原型/材质编辑页面.html`

---

## 页面流程

```
资产库入口 → 选择设备 → 表格操作 → 点击"编辑"按钮
                                        ↓
                              设备编辑页面 → 选择部件节点
                                        ↓
                              右侧修改材质属性
                                        ↓
                              底部选择材质拖拽应用
```

## 验收标准

| 验收项 | 标准 |
|--------|------|
| 结构验收 | 菜单、表格、编辑页面结构完整 |
| 布局验收 | 工具栏位于三维场景区内部 |
| 视觉验收 | 现代、清晰、重点突出 |
| 交互验收 | 具备 tab、高亮、选中、按钮等演示效果 |
| 流程验收 | 资产库→编辑页面流程贯通 |
